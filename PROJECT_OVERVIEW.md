# Government Job Portal
## MERN Stack - Full-Stack Application

A comprehensive government job notification portal built with React, Express, MongoDB, and Node.js.

### What's Included

✅ **Complete Backend (Express.js)**
- RESTful API with all CRUD operations
- Secure JWT authentication
- MongoDB integration
- File upload handling with Multer
- Password hashing with bcryptjs
- CORS configuration

✅ **Complete Frontend (React.js)**
- Modern React components
- React Router for navigation
- Axios for API calls
- Context API for state management
- Responsive CSS styling
- Mobile-first design

✅ **Key Features**
- Public job listings with search and filter
- Admin login with secure authentication
- Add, edit, delete job notifications
- PDF upload and download functionality
- Responsive design (mobile, tablet, desktop)
- Job deadline alerts
- Department filtering
- Direct application links

### Quick Start

1. **Backend:**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

2. **Frontend (new terminal):**
   ```bash
   cd frontend
   npm install
   npm start
   ```

3. **Admin Credentials:**
   - Username: `admin`
   - Password: `admin@123`

### Technology Stack

- **Frontend:** React, React Router, Axios, CSS3
- **Backend:** Node.js, Express.js, MongoDB, Multer
- **Authentication:** JWT, bcryptjs
- **Database:** MongoDB

### Project Structure

- `/backend` - Express server with routes, models, controllers
- `/frontend` - React application with components and pages
- `/backend/uploads` - PDF file storage
